# Fitness Calorie Predictor (ML Project)

##  Description
This project predicts calories burned based on distance, time, weight, height, BMI, and activity type using Machine Learning.

##  ML Model
- Linear Regression
- Supervised Learning

##  Activity Types
1 = Walking  
2 = Running  
3 = Cycling  

##  How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python main.py

##  Inputs
- Distance (km)
- Time (minutes)
- Weight (kg)
- Height (meters)
- Activity type

##  Output
- Estimated calories burned
- BMI
- Model accuracy
